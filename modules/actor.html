Module actor

This namespace contains all functions added to an actor's spheres property, where they act like regular methods available on the actor.

Additionally, the namespace contains types related to an actor's system.spheres property, documenting which properties are available and their types. For a list of all paths the module is aware of, see SpheresRollDataPaths.

Example: Rolling a Magic Skill Check

const actor = game.actors.get("some-actor-id");
await actor.spheres.rollMsb();
