Module hookEvents

Hooks

This namespace contains all the hooks fired by this module. All hooks introduced by the module are prefixed with a pf1spheres scope. Each hook contains a remark denoting whether it is called with Hooks.callAll or Hooks.call. Only hooks that are called with Hooks.call can be stopped by returning false from the callback.
